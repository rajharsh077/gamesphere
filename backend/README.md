# GameSphere Backend

This folder contains the backend implementation for GameSphere.
The architecture uses Express, Mongoose, JWT authentication, Socket.IO, and a Controller-Service-Repository pattern.

## Planned structure
- `src/config`
- `src/controllers`
- `src/services`
- `src/repositories`
- `src/models`
- `src/routes`
- `src/middleware`
- `src/sockets`
- `src/validators`
- `src/utils`

Phase 1 is architecture/design only. Implementation starts after approval.
