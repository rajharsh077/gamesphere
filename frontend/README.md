# GameSphere Frontend

This folder contains the frontend implementation for GameSphere.
The architecture uses React, Vite, Tailwind CSS, Redux Toolkit, React Router, Axios, and Socket.IO Client.

## Planned structure
- `src/components`
- `src/pages`
- `src/store`
- `src/store/slices`
- `src/services`
- `src/routes`
- `src/sockets`
- `src/styles`
- `src/assets`
- `src/utils`

Phase 1 is architecture/design only. Implementation starts after approval.
